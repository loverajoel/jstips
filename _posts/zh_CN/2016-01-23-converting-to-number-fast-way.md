将字符串转换为数字是极为常见的。最简单和快速的方法([jsPref](https://jsperf.com/number-vs-parseint-vs-plus/29))`+`(加号) 来实现。

```javascript
var one = '1';

var numberOne = +one; // Number 1
```

你也可以用`-`(减号)将其转化为负数值。

```javascript
var one = '1';

var negativeNumberOne = -one; // Number -1
```
